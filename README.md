# Abnamro Assessment 

## Setup instruction

#### Prerequisite
- Git 2.x
- Maven 3.x
- Java 11

Clone repository
```bash
git clone https://github.com/razi-ahmad/cooking-instruction.git
```

Run test cases
  ```bash
  mvn clean test
  ```

Package project
  ```bash
  mvn package
  ```

Run application
  ```bash
  java -jar target/cooking-instruction-0.0.1-SNAPSHOT.jar
  ```
